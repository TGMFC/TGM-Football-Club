# The Talking Game Masters FC – Official PWA

🏆 A lightweight, mobile-first, **installable Progressive Web App (PWA)** for Eastside United & The Talking League fans in South Africa.

Built with pure HTML, CSS, and vanilla JavaScript — **no frameworks, fast loading** even on 3G 📶  
Perfect for Android users to add to home screen and never miss a fixture, news, or RSVP.

---

## 🧱 Features
- ✅ Responsive design (phones, tablets, desktop)
- 🎨 Black & Yellow theme (TGM FC colors)
- 📲 Installable PWA (via `manifest.json` + `service-worker.js`)
- ⚽ Fixtures display (loaded from JSON)
- 📝 RSVP form with WhatsApp-ready confirmations
- 🌐 Hosted free via GitHub Pages
